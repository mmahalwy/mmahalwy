Today's founder session in YC was sharing our startup stack and our tech stack. I thought I'd share ours more publically.

## Startup stack

- Banking: [Brex](https://brex.com)
- Shares and certificate management: [Carta](https://carta.com)
- Product management: [Linear.app](https://linear.app)
- Data management: [Airtable](https://airtable.com)
- Design: [Figma](https://figma.com)
- Documents: [Notion](https://notion.so)
- Communication: [Slack](https://slack.com)
- Email: [Google](https://google.com) and [Superhuman](https://superhuman.com)
- Calendar: [Calendly](https://calendly.com)
- Music: [Spotify](https://spotify.com)
- Source control: [Github](https://github.com)
- Code editor: [VS Code](https://vscode.com) with [Tabnine](https://tabnine.com)
- Asset management: [Filestack](https://filestack.com) and [Uploadcare](https://uploadcare.com)
- Mass emails: [Sendgrid](https://sendgrid.com)

## Tech stack

- Frontend deployment and hosting: [Vercel/Nextjs](https://vercel.com)
- Frontend stack: [Nextjs](https://nextjs.com), React, [Ant Design](https://ant.design/), Apollo GraphQL
- API deployment: [Heroku](https://heroku.com)
- API stack: Nodejs, Koa, Sequelize ORM, GraphQL
- Error tracking: [Sentry](https://sentry.com)
